⚡ STM32 Smart Baton - 智能手持 BMS 与动态交互终端本项目是一个基于 STM32F103C8T6 微控制器的手持电棒/高功率设备智能外设终端。它集成了 BMS 电池状态监测、环境温度感知、人体雷达探测 以及基于 U8g2 的 高帧率粒子动画系统。通过对硬件电压的敏锐嗅探与平滑滤波，系统能够实时无缝反馈充放电状态，并提供极具科技感的视觉交互体验。✨ 核心功能特性🔋 智能电池状态机与 BMS 监测实时电压与百分比：基于 2S 锂电池 (6.0V - 8.4V) 充放电曲线的非线性 SoC (State of Charge) 查找表，精准估算电量。SMA-3 & IIR 级联滤波：结合 3 次滑动平均 (SMA) 与无限脉冲响应 (IIR) 滤波算法，有效剔除大功率放电带来的电压瞬态噪声。充放电状态自适应：通过捕捉极短时间内的电压突变（暴跌即放电，稳幅攀升即充电），无需外部触发引脚即可自动切换 UI 状态。🎨 高级动态 UI 与粒子系统 (基于 U8g2)全屏闪屏动画 (Splash Screen)：开机自带流畅的随机散射粒子动画。物理引力粒子效果：待机状态下，主界面拥有受“重力”与“引力区”影响的游离粒子动画。放电雷电特效：侦测到高功率放电时，自动触发高频震动的“闪电”几何形变动画。智能充电预估：充电时展示专属吸引力粒子，并根据充入时间推算剩余充电时间 (ETA) 与预估内阻电压。🌡️ 实时环境与安全感知无阻塞温度读取：使用非阻塞式状态机驱动 DS18B20 温度传感器，保证 UI 渲染不卡顿（支持 -55°C 至 125°C）。附近雷达检测：集成 PIR 人体红外感应模块，当检测到附近有活体移动时，屏幕会动态浮现“小人”雷达预警图标。🛠️ 硬件规范与引脚分配本项目的核心运算由 STM32F103C8T6 (ARM Cortex-M3 @ 72MHz) 承担，系统工作电压为 3.3V。引脚接线图 (Pin Allocation)功能模块信号描述MCU 引脚外设引脚 / 定时器/状态机架构，确保极高的响应速度与系统可靠性。非线性 SoC 查表：soc_table[] 数组预置了 2S 18650 电池的放电特征曲线（6.6V/10% -> 7.4V/50% -> 8.2V/90%），解决了锂电池中间电压平缓导致的电量显示不准问题。多重去抖动状态机：针对电容充电、瞬间击发拉低电压等工况，代码中内置了 CHARGE_LOCKOUT_AFTER_DISCHARGE_MS 等防抖逻辑，防止设备在剧烈放电后误判为插入充电器。硬件级 SPI 加速：修改了 U8g2 底层驱动 u8x8_byte_4wire_hw_spi，通过 STM32 硬件 SPI1 直连 DMA / 寄存器，极大压榨了屏幕刷新率极限。🚀 编译与烧录部署环境准备：推荐使用 STM32CubeIDE 或 Keil MDK-ARM。依赖库：需克隆并包含最新的 U8g2 库。时钟配置：代码中已配置 HSE
