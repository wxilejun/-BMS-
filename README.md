# ⚡ STM32 智能手持辅助系统电棒 (Smart Baton BMS)

## 📖 项目简介

这是一个基于 STM32F103C8T6 微控制器打造的极客 DIY 项目——专为手持高功率设备设计的“智能仪表盘”。它不仅能精准监测电池的充放电状态，还具备极其酷炫的高帧率粒子动画、环境温度检测以及防近身的人体雷达侦测系统。非常适合想要为自己的硬核 DIY 装备增加互动感与科技感的玩家！

---

## ✨ 功能亮点

* **🔋 丝滑的电池管家**
摆脱粗糙的电压跳动！系统内置了平滑算法与真实的电池曲线预估，无论是大功率放电还是插入充电器，屏幕都能精准、稳定地反馈当前的电量百分比与实时电压。
* **🎨 极客风动态交互 UI**
* **开机动画**：启动自带流动的散落粒子特效。
* **待机界面**：屏幕中拥有受物理引力影响的游动粒子，拒绝死板。
* **工作特效**：侦测到高功率放电（击发）时，立刻无缝切入专属的“闪电”高频震动特效。
* **智能充电**：插入充电器后，动态粒子自动向中心汇聚，并实时推算当前充满所需的时间（ETA）。


* **🌡️ 环境感知与雷达预警**
实时读取设备所处环境的温度；当集成的人体红外雷达检测到周围有活体靠近时，屏幕会自动浮现雷达预警“小人”图标。

## 🛒 材料清单 (BOM)

开始组装前，你需要准备以下核心硬件：

1. **主控板**：STM32F103C8T6 最小系统板（俗称 Blue Pill）。
2. **显示屏**：0.96英寸 OLED 屏幕（**注意：务必购买支持 4 线 SPI 接口的版本**，以支撑项目的高帧率动画）。
3. **温度传感器**：DS18B20 模块（引脚处需自备一颗 4.7kΩ 上拉电阻）。
4. **人体感应模块**：微型 PIR 红外雷达感应器（如 HC-SR505 或 HC-SR501）。
5. **电压采样元件**：2 颗贴片或色环电阻（推荐 36kΩ 和 10kΩ，用于将电池高压安全分压给主控芯片）。

## 🔌 硬件接线指南

请按照下表将各个外设模块正确连接至 STM32 主控板：

| 模块名称 | 模块引脚 | 连接至 STM32 引脚 | 备注说明 |
| --- | --- | --- | --- |
| **OLED 屏幕 (SPI)** | CS | **PA2** | 片选信号 |
|  | DC | **PA3** | 数据/命令切换 |
|  | RES | **PA4** | 屏幕复位 |
|  | D0 / SCK | **PA5** | SPI 时钟 |
|  | D1 / SDA | **PA7** | SPI 数据 |
| **温度传感器** | DQ | **PA1** | 单总线数据（需接 4.7k 上拉至 3.3V） |
| **人体雷达 (PIR)** | OUT | **PB9** | 触发输出信号 |
| **电池电压检测** | - | **PA0** | 连接至电阻分压网络的中心采样点 |
| **系统供电** | VCC / VDD | **3.3V** | 给单片机及各大传感器供电 |
|  | GND | **GND** | 整个系统必须共地 |

> **💡 接线避坑提示**：电池的正极（最高可能达 8.4V）**绝对不能**直接连到 STM32 的 PA0 引脚，必须经过 36kΩ 与 10kΩ 电阻串联分压后，从中间引出信号接入单片机，否则会烧毁主控！

## 🚀 快速部署与测试

**第一步：检查硬件连线**
严格按照上方的表格完成所有硬件连接，仔细检查正负极与分压电路是否短路。

**第二步：烧录固件**

1. 下载本项目 Release 页面提供的最新固件。
2. 准备一个 ST-Link V2 或 DAP-Link 下载器，连接主控板的 SWD 接口（`3.3V`, `GND`, `SWDIO`, `SWCLK`）。
3. 使用 STM32CubeProgrammer 软件将固件一键刷入。

**第三步：开机把玩**
上电后，系统会自动跑完开机动画进入主页面：

* **测温**：用手捏住 DS18B20 探头，观察屏幕中间的温度数值上升。
* **测雷达**：在 PIR 感应器前方挥挥手，查看顶部是否立刻弹出警告小人。
* **测充电**：接通系统充电线，系统会侦测到电压稳步上升，随后自动切入充电预估倒计时界面。

## ⚠️ 安全与免责声明

* **高压危险**：本开源项目代码仅涉及 **监测与显示终端** 的控制。如果你将此系统与高压发生器、升压模块等危险设备联动，请务必具备相关的强电操作常识并做好绝缘保护。
* **电池安全**：系统默认支持 2S 动力锂电池直驱。大功率充放电环境下，请**务必在硬件层面**为电池加装过充/过放/短路保护板，本系统的软件监测不能替代物理级的安全保护。
* **合理使用**：本项目仅供极客 DIY 爱好者学习、交流单片机技术与 UI 动效开发。开发者不对任何因操作不当引发的硬件损坏及人身安全问题负责。
